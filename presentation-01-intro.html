Part 1 · 개념

HTML이란?

HTML은 "태그"로 내용을 감싸서
브라우저에게 알려주는 언어입니다.

<h1> → 큰 제목

<p> → 문단 (글)

<img> → 이미지

<a> → 링크

<!-- 이것이 HTML 코드입니다 -->
<h1>안녕하세요!</h1>
<p>제 홈페이지입니다.</p>
<img src="photo.jpg">

💡 태그는 항상 < >로 감싸고, 대부분 <열기>내용</닫기> 쌍으로 사용합니다.

5 / 22
Part 1 · 개념

CSS란?

CSS는 HTML에 스타일을 입히는 언어입니다.

color → 글자 색상

font-size → 글자 크기

background → 배경 색상

margin → 바깥 여백

/* 이것이 CSS 코드입니다 */
h1 {
  color: blue;
  font-size: 32px;
}

p {
  color: gray;
}

실습 3: 코드를 파일로 저장하기

1

코드 복사

Claude 응답의 코드 블록 오른쪽 위 "Copy" 버튼 클릭

2

메모장 열기

Windows: 메모장 (시작 → "메모장" 검색)

붙여넣기 & 저장

Ctrl+V (붙여넣기) → 파일 → 다른 이름으로 저장

4

더블 클릭으로 열기

저장한 파일을 더블클릭 → 브라우저에서 홈페이지 확인!

실습 3

Claude가 만든 코드 살펴보기

<!DOCTYPE html>
<html>
<head>
  <title>내 홈페이지</title>
  <style>
    body { background: #f0f0f0; }
    h1 { color: navy; }
  </style>
</head>
<body>
  <h1>홍길동</h1>
  <p>취미: 독서</p>
</body>
</html>

<head>

→ 페이지 설정 (제목, 스타일)

<style>

→ CSS가 들어가는 곳

<body>

→ 실제 보이는 내용

<h1> <p>

→ 제목과 글 내용

Part 4 · 팁

좋은 요청(프롬프트) 작성법

❌ 나쁜 예

"홈페이지 만들어줘"

→ 너무 막연해서 원하는 결과가 안 나옴

✅ 좋은 예

"카페 소개 홈페이지를 만들어줘. 메뉴판, 위치, 영업시간을 포함하고, 따뜻한 갈색 톤으로 디자인해줘."

✨ 좋은 프롬프트의 4요소

1. 목적 — 무엇을 위한 페이지인가?

2. 내용 — 어떤 정보가 들어가는가?

3. 스타일 — 어떤 분위기/색상?

4. 형식 — 하나의 HTML 파일로!

따라해볼 수 있는 프롬프트 예시

자기소개

"이름이 '김민수'이고, 대학생이고, 취미는 사진 찍기야. 깔끔하고 모던한 자기소개 홈페이지를 하나의 HTML 파일로 만들어줘. 파란색 계열로 해줘."

소규모 비즈니스

"동네 꽃집 '봄날의 꽃' 홈페이지를 만들어줘. 꽃 종류 소개, 가격표, 찾아오시는 길, 전화번호를 넣어줘. 분홍색과 초록색 톤으로 예쁘게 디자인해줘."

취미 블로그

"내 여행 사진을 모아놓은 블로그를 만들어줘. 사진 갤러리 형태로 하고, 여행지 이름과 간단한 설명을 넣어줘. 감성적인 분위기로 해줘."

자주 겪는 문제 & 해결법

😱 파일이 코드로 보여요!

→ 파일 확장자가 .html인지 확인하세요. .txt로 저장하면 코드가 그대로 보입니다.

😕 한글이 깨져요!

→ Claude에게 "코드 맨 위에 <meta charset="UTF-8"> 넣어줘"라고 요청하세요.

🤔 결과가 마음에 안 들어요!

→ 구체적으로 수정 요청하세요. "제목 색을 빨간색으로 바꿔줘" 처럼 하나씩!

📋 코드가 너무 길어요!

→ "Copy" 버튼으로 전체 복사. 일부만 복사하면 깨질 수 있어요.
