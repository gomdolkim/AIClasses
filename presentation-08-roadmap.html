학습 전략

AI와 함께 배우는 5가지 방법

1. 코드 설명 요청 — "이 코드가 뭐하는 건지 한 줄씩 설명해줘"

2. 에러 해결 — 에러 메시지를 복사해서 "이 에러 왜 나오는지 알려줘"

3. 리팩토링 요청 — "이 코드를 더 좋게 고쳐줘. 왜 그렇게 하는지 설명도 해줘"

4. 개념 비유 요청 — "async/await를 식당 비유로 설명해줘"

5. 퀴즈 요청 — "JavaScript 기초 퀴즈 10문제 내줘" → 실력 점검!

주의

초보자가 빠지기 쉬운 함정

❌ 함정 1: 튜토리얼 수집가

다 알아야 시작한다고 생각 → 60%만 알아도 만들기 시작!

❌ 함정 4: AI에 100% 의존

복사만 하고 이해 안 함 → 코드를 한 줄씩 이해하기!
