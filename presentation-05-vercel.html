GitHub 실습

실습: GitHub 회원가입

1

github.com 접속

2

GitHub 실습

실습: 저장소(Repository) 만들기

1

+ 버튼 → "New repository"

오른쪽 상단의 + 아이콘 클릭

2

이름 입력

Repository name: my-website (영문, 소문자, 하이픈)

3

Public 선택 → Create

💡 저장소(Repository) = 프로젝트 폴더. 줄여서 "레포(repo)"라고 부릅니다

25 / 28
문제 해결

자주 겪는 문제 & 해결법

😱 404 페이지가 나와요!

→ 파일 이름이 index.html인지 확인. Index.html, INDEX.HTML은 안 됩니다.

😕 GitHub 레포가 안 보여요!

→ Vercel 설정에서 "Adjust GitHub App Permissions" 클릭 → 레포 접근 권한 허용

🤔 도메인 연결했는데 안 돼요!

→ DNS 변경은 최대 48시간 걸릴 수 있어요. 보통 10분~1시간이면 됩니다.

📋 수정했는데 반영이 안 돼요!

→ Vercel 대시보드에서 배포 상태 확인. 브라우저 캐시 삭제(Ctrl+Shift+R)도 시도하세요.
