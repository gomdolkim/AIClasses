플러그인 설치

연동 설치하기 (3단계)

1

설정 페이지 열기

Claude.ai 왼쪽 하단 → 이름 클릭 → Settings

2

Integrations 탭 선택

왼쪽 메뉴에서 "Integrations" 클릭

3

원하는 서비스 연결

서비스 선택 → "Connect" → 해당 서비스 로그인 → 권한 허용

✅ 연결 완료 후 Claude에게 "내 Google Drive에서 ○○ 파일 찾아줘" 같은 요청 가능!

11 / 28
모델

언제 어떤 모델을 쓸까?

⚡ Haiku → "이 단어 뜻이 뭐야?", "이 문장 영어로 번역해줘"

🎯 Sonnet → "홈페이지 만들어줘", "이 코드 설명해줘", "보고서 써줘"

🧠 Opus → "이 논문 깊이 분석해줘", "복잡한 시스템 설계해줘"

💡 큰 모델 = 더 똑똑하지만 느리고 비쌈. 간단한 일에 Opus는 과한 투자!

에이전트의 활용 예시

🛡️ 보안 에이전트

코드에서 보안 위험을 찾아내서 경고

에이전트는 주로 Claude Code (개발자 도구)에서 사용됩니다

보너스
스타일
Styles
Claude의 말투와 형식을 조절하는 설정

🎭 설정 방법

1. Claude.ai 하단 → 모델 선택 옆의 스타일 아이콘 클릭

2. 기본 제공 스타일 중 선택하거나 직접 만들기

3. "간결하게", "친근하게", "전문적으로" 등 원하는 톤 설정

💡 좋아하는 글을 붙여넣고 "이 말투로 대답해줘"라고 하면 스타일이 만들어져요!
